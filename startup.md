# راهنمای نصب و راه‌اندازی در ویندوز (اجرای دستی)

این راهنما مراحل نصب و اجرای دستی سیستم مانیتورینگ اینترنت اشیاء صنعتی (Industrial IoT Monitoring System) را روی یک سیستم ویندوزی واقعی قدم‌به‌قدم توضیح می‌دهد.

## ۱. نصب Docker Desktop

۱. نرم‌افزار Docker را از لینک زیر دانلود کنید:
[Docker 4.52.0 - Windows](https://www.yasdl.com/309404/%D8%AF%D8%A7%D9%86%D9%84%D9%88%D8%AF-docker.html)
۲. نسخه ویندوز را نصب کنید.
۳. بعد از پایان نصب، برنامه Docker Desktop را باز کنید.
۴. صبر کنید تا موتور داکر به‌طور کامل اجرا شود (آیکون داکر در گوشه پایین سمت راست ویندوز باید وضعیت Running را نشان دهد).
۵. اگر با خطایی مواجه شدید، سیستم خود را یک بار ری‌استارت کنید.

## ۲. نصب Node.js و npm

۱. نرم‌افزار Node.js را از لینک زیر دانلود کنید:
[Node.js 26.2.0 Win/Mac](https://www.yasdl.com/176044/%D8%AF%D8%A7%D9%86%D9%84%D9%88%D8%AF-node-js.html)
۲. حتماً نسخه `LTS` را نصب کنید.
۳. در طول مراحل نصب، هیچ‌یک از تنظیمات پیش‌فرض را تغییر ندهید.
۴. بعد از نصب، یک پنجره PowerShell باز کنید و دستورات زیر را وارد کنید:

```powershell
node -v
npm -v
```

۵. اگر شماره نسخه‌ها نمایش داده شد، یعنی نصب با موفقیت انجام شده است.

## ۳. رفتن به پوشه پروژه

۱. پوشه پروژه را در سیستم ویندوزی خود کپی کنید.
۲. پنجره PowerShell را باز کنید.
۳. با استفاده از دستور `cd` وارد پوشه اصلی پروژه (پوشه‌ای که فایل `docker-compose.yml` در آن قرار دارد) بشوید:

```powershell
cd C:\Path\To\sensor-data-recorder
```

## ۴. اجرای سرویس‌های Docker

۱. در حالی که داخل پوشه پروژه هستید، دستور زیر را برای اجرای سرویس‌های پس‌زمینه (InfluxDB، Mosquitto، Node-RED) وارد کنید:

```powershell
docker compose up -d
```

_(اگر دستور `docker compose` شناخته نشد، از `docker-compose up -d` استفاده کنید)_
۲. چند ثانیه صبر کنید تا کانتینرها ساخته و اجرا شوند.

## ۵. نصب پکیج‌های Node

۱. در همان پنجره PowerShell، وارد پوشه `sensor-app` بشوید:

```powershell
cd sensor-app
```

۲. پکیج‌های موردنیاز (مثل `mqtt` و `influxdb-client`) را نصب کنید:

```powershell
npm i
```

## ۶. اجرای برنامه اصلی

۱. در حالی که هنوز داخل پوشه `sensor-app` هستید، اسکریپت اصلی را اجرا کنید:

```powershell
node index.js
```

۲. حالا لاگ‌ها را به‌صورت زنده در ترمینال خود مشاهده خواهید کرد و داده‌هایی که از سنسورها یا پنل HMI واقعی ارسال می‌شوند، دریافت و در دیتابیس InfluxDB ذخیره می‌گردند.

## ۷. اتصال سنسور واقعی یا پنل HMI

۱. سنسور فیزیکی یا پنل HMI شما باید داده‌ها را به بروکر MQTT ارسال کند.
۲. اگر پنل HMI روی یک دستگاه دیگر قرار دارد، در تنظیمات MQTT پنل، باید **آدرس IP** این سیستم ویندوزی را وارد کنید.
۳. **تنظیمات MQTT:**

- Host (آدرس آی‌پی): `192.168.x.x` (آدرس آی‌پی سیستم ویندوز شما)
- Port (پورت): `1883`
- Username (نام کاربری): _(خالی بگذارید)_
- Password (رمز عبور): _(خالی بگذارید)_

## ۸. متوقف کردن سیستم

۱. برای متوقف کردن برنامه Node، در همان پنجره PowerShell که `node index.js` در حال اجراست، کلیدهای زیر را فشار دهید:

```text
Ctrl + C
```

۲. برای متوقف کردن سرویس‌های داکر، یک پوشه به عقب برگردید (پوشه اصلی پروژه) و دستور زیر را اجرا کنید:

```powershell
cd ..
docker compose down
```
